## Bắt đầu

Đầu tiên, cài đặt gói:

```bash
conmposer install
```

## Chạy dự án

Trong thư mục dự án, bạn có thể chạy:

```bash
php artisan server
```

Chạy ứng dụng ở môi trường local.\
Open [http://localhost:8080](http://localhost:8080) để xem nó trong trình duyệt của bạn.
